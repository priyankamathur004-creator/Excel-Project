# Bare International Style Advisor Performance Analysis

# Overview

Project evaluates Style Advisor performance using customer journey criteria.

Data includes store ambiance, first impressions, discovery, trial experience, objection handling, closure, and overall service.

Focus: Identify high-performing regions, highlight gaps, and recommend improvements.

# Objectives
Performance Grading → Categorize advisors into High, Average, Low, Bottom performers.

Pivot Table Analysis → Compare average evaluation scores by zone.

High Performer Identification → Find regions with the highest concentration of top performers.

Visualization → Use charts (bar, pie, line, heatmap, stacked bar) to highlight trends.

Presentation Report → Summarize findings and provide actionable recommendations.

# Dataset Information
Evaluation_ID, Date, Score → Core evaluation metrics.

Criteria Scores → Store ambiance, impressions, discovery, trial, recommendations, closure.

Location Data → City, state, zone, country.

Visit Details → Entry/exit times, duration, number of advisors/customers present.

Auditor Data → Age, gender, attire.

Style Advisor Details → Names and descriptions.

# Key Insights
North Zone → Consistently high average scores, stable standards.

East Zone → Highest concentration of High Performers (50%), benchmark for best practices.

South Zone → Lagging performance, requires immediate intervention.

West Zone → Average cluster, similar to East but with fewer high performers.

Majority of advisors fall into Average category, with ~13% in Bottom tier needing targeted training.

# Business Value
Provides clear visibility into advisor performance across regions.

Identifies success clusters for knowledge sharing.

Highlights underperforming zones for targeted coaching.

Supports management decisions to improve customer experience and sales outcomes.
